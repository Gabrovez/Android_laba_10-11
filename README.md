## Список студентов (Student Material Design)

### 1. Название проекта
**Список студентов**

Приложение для отображения списка студентов с демонстрацией использования принципов Material Design.

### 2. Цель работы
Научиться применять принципы Material Design в Android-приложениях с использованием Jetpack Compose. В результате работы создано приложение, демонстрирующее использование цветовых схем, типографики, форм и анимаций Material Design.

### 3. Функциональность
*   Отображение списка студентов (аватар, имя, возраст).
*   Поддержка светлой и тёмной темы оформления.
*   Использование кастомной цветовой схемы (Primary, Secondary, Tertiary).
*   Применение кастомных форм (круглые аватарки, скруглённые углы карточек).
*   Использование кастомной типографики (шрифты Montserrat и Abril Fatface).
*   Наличие верхней панели приложения (Top App Bar) с логотипом и названием.
*   Анимация раскрытия и сворачивания подробного описания студента.

### 4. Технологии
*   Язык программирования: Kotlin
*   UI Toolkit: Jetpack Compose
*   Дизайн-система: Material Design 3
*   Среда разработки: Android Studio
*   Контроль версий: Git, GitHub
*   Библиотеки:
    *   androidx.compose.material:material-icons-extended
